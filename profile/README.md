
# ZETA

## Sobre o aplicativo
ZETA é um aplicativo que oferece treinamentos para produtores do setor pecuário, permitindo o acesso a cursos sobre diversos segmentos e áreas de atuação.

Para produtores vinculados a empresas, também disponibilizamos um perfil corporativo, no qual fornecedores podem acompanhar o progresso de cada colaborador por meio de gráficos intuitivos e detalhamento sobre o desempenho de cada um.

## Principais funcionalidades
Para o perfil do fornecedor:
- Visualização do desempenho geral
- Ranking dos produtores por pontos
- Visualização do progresso individual

Para o perfil do produtores:
- Visualização do desempenho por curso
- Metas diárias de cada curso
- Chatbot para tirar dúvidas sobre o conteúdo
- Chatbot para indicação de curso baseado no tópico

## Linguagens utilizadas
Para o desenvolvimento desse projeto, utilizamos as seguintes linguagens:
- DAX
- HTML e CSS
- Java
- JavaScript
- Python
- PL/pgSQL
- Redis Command Language (RCL)
- MongoDB Query Language (MQL)

## Como utilizar
1. Baixe o aplicativo
2. Crie uma conta como *produtor* ou *fornecedor*
3. Explore os cursos e visualize seu desempenho ou o desempenho de seus produtores.
